# slide-presentation-mac-lenin

## Demo Three.js + bàn tay (`flag-hands.html`)

Trang độc lập: [flag-hands.html](flag-hands.html) (mở qua **localhost** hoặc **HTTPS** để trình duyệt cho phép camera).

- **Nắm đấm** trước webcam: các điểm sáng tụ thành lá cờ Việt Nam (đỏ + vàng).
- **Mở bàn tay**: hạt tản ra và bay quanh; có lực đẩy nhẹ theo vị trí cổ tay.
- **Hất tay ngang nhanh**: thêm xung lực ngang cho hạt (nhận diện vận tốc cổ tay).

Cần tải model MediaPipe (~vài MB) lần đầu; GPU lỗi sẽ tự thử CPU.
